# NoteGround

Installable Obsidian plugin bundle for NoteGround.

This repository contains the published plugin artifacts only:

- `main.js`
- `manifest.json`
- `styles.css`

Source code lives in the main development repository:

- https://github.com/dangquan1402/obsidian-grounded

## Install

1. Download `main.js`, `manifest.json`, and `styles.css` from this repository or its releases.
2. Create the folder `<your-vault>/.obsidian/plugins/note-ground/`.
3. Copy the three files into that folder.
4. Open Obsidian and enable **NoteGround** in **Settings > Community plugins**.

## Requirements

Install at least one supported AI CLI locally:

- Claude Code
- OpenAI Codex
- Gemini CLI
